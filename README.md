# AI Text Humanizer Extension

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/aceyTwenty1/ai-text-humanizer-extension.git
   ```
2. Navigate to the project directory:
   ```
   cd ai-text-humanizer-extension
   ```
3. Install the required dependencies:
   ```
   npm install
   ```

## Usage

To use the AI Text Humanizer Extension, follow these steps:

1. Import the extension in your project:
   ```javascript
   import { humanizeText } from 'ai-text-humanizer';
   ```
2. Call the function with your text:
   ```javascript
   const originalText = "Your text here";
   const humanizedText = humanizeText(originalText);
   console.log(humanizedText);
   ```
